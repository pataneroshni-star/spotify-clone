# Spotify Clone 🎵

A Spotify-inspired music player UI built using HTML and CSS.

## Features

- Responsive Sidebar
- Music Player Footer
- Playlist Cards
- Spotify-like User Interface
- Modern Layout using Flexbox

## Technologies Used

- HTML5
- CSS3
- Font Awesome
